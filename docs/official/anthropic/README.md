# Anthropic Documentation Repository

## Purpose
Source truth documentation for Claude AI, API references, and Anthropic best practices.

## Structure
```
anthropic/
├── api/              # Claude API documentation
├── best-practices/   # Usage patterns and guidelines
```

## Usage
This directory contains official Anthropic documentation to reference during development:

2. **Best Practices** - Follow official guidelines for optimal performance
3. **Model Specs** - Understand capabilities and limitations
4. **Safety Guidelines** - Ensure compliant and safe AI usage
